# What is WebSocket
## Socket
 - 클라이언트와 서버 간의 연결
 - 실시간으로 데이터 송수신
## Server
- Client와의 Socket 연결을 설정하고 관리
- Client로 이벤트를 방출
- Client로부터 이벤트를 수신
## Client
- Server와의 Socket 연결을 설정하고 관리
- Server로 이벤트를 방출
- Server로부터 이벤트를 수신
